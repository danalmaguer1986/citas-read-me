Base de Datos con flyway-8.5.1

- Abrir carpeta "flyway-8.5.1" que está en:
	 "CitasMedicas\02.Construcción"
- Crear Base de Datos en SQL Server destino
- Cambiar conexión a SQL Server en archivo "flyway.conf"
  que está en carpeta config.
- Dentro de "flyway-8.5.1" ejecutar command prompt
- En consola, ejecutar "flyway migrate"

* En caso de que esta versión de Flyway no soporte la versión
  actual de SQL, descargar flyway más reciente y realizar
  los mismos pasos *
